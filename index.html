구글 시트 연동 설정
구글 스프레드시트를 웹에 게시하면, 이 대시보드가 자동으로 데이터를 읽어옵니다.
한 번 설정하면 시트를 업데이트할 때마다 새로고침만 하면 됩니다.
1
구글 시트 형식 맞추기
아래 표 형식에 맞게 시트를 작성하세요. 시트1에 월 기준 데이터, 시트2에 제품 데이터를 입력합니다.
2
웹에 게시
구글 시트 메뉴 → 파일 → 웹에 게시 → 각 시트를 CSV로 선택 → 게시 클릭. URL 두 개가 생성됩니다.
3
URL 입력 후 연결
생성된 URL을 아래에 붙여넣고 연결하세요.
